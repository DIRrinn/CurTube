## Dependencies:
*Python:*
- Python 3+

*Python modules*:
- os;
- pytubefix;
- ffmpeg-python (for converting);
- [Curses](https://docs.python.org/3/library/curses.html#module-curses)

## Issues

- Remove relative path. Do absolute. (-)

## Instructions

Don't forget to change output path lol

1. Clone repository
```
git clone https://github.com/DIRrinn/CurTube.git
```

2. Go to CurTube's directory

```
cd CurTube
```

3. Install dependencies

```
pip install pytubefix ffmpeg-python
```
*There's no need to install `os` and `curses` modules*

4. Execute `python tui.py`
